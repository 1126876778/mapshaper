mapshaper

25 July, 2013

There is a new version of MapShaper online at mapshaper.org.

Features include:

  • A more reliable topology algorithm
  • Client-side processing -- your data stays private
  • A command line script for non-interactive simplification
  • Output in Shapefile, GeoJSON and TopoJSON formats
  • Open source

The project is hosted on github, where you can download the source code, leave feedback and find instructions on using the command line script.

The original version is still available here.
